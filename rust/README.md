# NullSec HashCrack

Multi-threaded hash cracker supporting multiple algorithms.

## Features

- MD5, SHA1, SHA256, SHA512 support
- Dictionary attack mode
- Fast multi-threaded performance
- Part of the NullSec security toolkit

## Installation

```bash
cargo install nullsec-hashcrack
```

## Usage

```bash
nullsec-hashcrack <hash> <wordlist> [algorithm]
```
