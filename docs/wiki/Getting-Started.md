# Getting Started

## Installation

```bash
git clone https://github.com/bad-antics/nullsec-tools
cd nullsec-tools
./install.sh
```

## Usage

Each tool has its own directory with a README and usage instructions.

```bash
# List available tools
ls tools/

# Run a specific tool
cd tools/scanner
./run.sh --help
```

## Requirements

- Linux (Ubuntu 22.04+ or similar)
- Python 3.9+
- Go 1.21+ (for Go-based tools)
- Rust 1.70+ (for Rust-based tools)
