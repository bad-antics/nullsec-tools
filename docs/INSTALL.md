# NullSec Tools Installation

## Quick Install
```bash
git clone https://github.com/bad-antics/nullsec-tools
cd nullsec-tools
./install.sh
```

## Manual Install
```bash
git clone https://github.com/bad-antics/nullsec-tools
cd nullsec-tools
pip install -r requirements.txt
```

## Verify
```bash
nullsec-tools --version
nullsec-tools list
```
